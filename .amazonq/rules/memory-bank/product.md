# WashBox - Product Overview

## Project Purpose
WashBox is a comprehensive laundry management system designed to streamline laundry operations with integrated customer management, order tracking, real-time GPS tracking, and automated workflows for pickup and delivery services.

## Value Proposition
- **Operational Efficiency**: Automated order processing, route optimization, and staff task management
- **Customer Experience**: Real-time tracking, mobile app access, multiple payment methods, and push notifications
- **Business Intelligence**: Analytics dashboards, financial reports, and performance metrics
- **Scalability**: Multi-branch support with centralized admin control

## Key Features

### Core Functionality
- **Customer Management**: Registration, profiles, order history, ratings
- **Laundry Orders**: Create, track, manage orders with service customization
- **Pickup/Delivery System**: Schedule requests, real-time GPS tracking, route optimization
- **Payment Processing**: Multiple payment methods with secure transactions
- **Notifications**: Firebase FCM push notifications for order updates
- **Promotions**: Discount codes and promotional campaigns
- **Unclaimed Reminders**: Automated reminders on Day 3, 5, 7

### Admin Dashboard
- Real-time analytics and statistics
- Customer and order management
- Staff management and performance tracking
- Service and pricing configuration
- Route optimization and mapping
- Financial reports and revenue tracking
- System settings and configuration

### Staff Dashboard
- Pickup request management
- Order processing workflow
- Route navigation with map integration
- Customer communication tools
- Task management and assignment

### Mobile App
- RESTful API for iOS/Android apps
- Firebase authentication
- Push notifications
- Real-time order tracking
- In-app payment processing
- Location-based services

## Target Users
- **Laundry Business Owners**: Multi-branch management, analytics, financial control
- **Staff Members**: Order processing, route navigation, customer service
- **Customers**: Order placement, tracking, payment, ratings

## Use Cases
1. Customer places laundry order via mobile app
2. Staff receives notification and schedules pickup
3. Real-time GPS tracking during pickup/delivery
4. Order processing at laundry facility
5. Delivery with customer confirmation
6. Automated reminders for unclaimed items
7. Admin reviews analytics and generates reports
